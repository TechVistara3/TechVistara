ROUND 03
Code Chaos

No elimination round. Evaluates coding skills, logical thinking and debugging ability. Each member does exactly one activity.

i. Reverse Coding 2 MEMBERS

Participants will be given the expected program output on paper. Write the code that produces the given output and execute it.

TIME
15-20 mins
MEMBERS
2
ii. Debugging 1 MEMBER

A block of code with errors (syntax, logical, or runtime) will be given. Identify the errors, correct them, and execute the program.

TIME
10 mins
MEMBERS
1
iii. Blind Coding 1 MEMBER

A piece of code will be briefly displayed on screen. Participants must retype the code accurately after the display is turned off.

TIME
10 mins
MEMBERS
1
IMPORTANT RULE

Participants cannot take part in more than one activity. Each team member must participate in only one coding activity.

ROUND 04
Hunting Quest

Tests attention to detail, analytical thinking and teamwork. Two exciting sub-rounds await!

i. Scavenger Hunt

A PDF document with information will be provided along with an MCQ sheet. All answers are hidden within the PDF.

TIME
30 mins
POINTS
1 / answer
  • Only one mobile phone per team allowed to view the PDF.
  • Search function is strictly not allowed.
ii. Treasure Hunt

A fun surprise round testing logic, decoding skills and teamwork! Teams receive a coded clue or puzzle to decipher and progress stage by stage.

  • Teams progress only after successfully decoding the given clue.
  • A hint will be provided every 5 minutes if stuck.
